This project is a modern web application that allows users to calculate their **Body Mass Index (BMI)** by entering height and weight information. Based on the calculation, personalized **dietary recommendations** are provided, and data is persisted in the browser using **localStorage**.

### üöÄ Features

- **Accurate BMI Calculation:** Real-time calculation using height and weight inputs.
- **Personalized Diet Plans:** Specific nutritional advice based on the result (Underweight, Normal, Overweight, Obese).
- **Data Persistence:** Integrated `localStorage` ensures data is kept even after the page is refreshed.
- **URL Synchronization:** Seamless navigation between "Calculator" and "What is BMI?" pages via React Router.
- **Modern UI:** Responsive and high-performance design built with `Styled Components`.

- **Frontend:** React 18+ (Vite)
- **Styling:** Styled Components
- **Routing:** React Router DOM
- **Hooks:** useState, useEffect (State & Lifecycle management)
- **Storage:** Browser LocalStorage API
